Bienvenue sur GEONIS Datashop !

GEONIS Datashop vous permet d’exécuter des requêtes de données sûres et simples.

Pour pouvoir acquérir des informations cartographiques, vous devez vous enregistrer dans le système comme utilisateur occasionnel ou permanent. Si vous ne disposez pas encore d’un compte d’utilisateur permanent, connectez-vous comme suit :

a.    S’il vous faut une information cartographique de manière occasionnelle, connectez-vous comme utilisateur occasionnel. En tant qu’utilisateur occasionnel, vous pouvez déposer 5 demandes en 2 semaines.

b.    S’il vous faut des informations cartographique de manière régulière, ouvrez un compte en tant qu’utilisateur permanent pour pouvoir déposer un nombre de demandes illimité. Cliquez sur « Nouvel enregistrement ».

Si vous êtes déjà inscrit en tant qu’utilisateur permanent et que vous avez oublié votre mot de passe, cliquez sur « Mot de passe oublié ? »

 

1. Connexion comme utilisateur occasionnel

Remplissez le formulaire d’inscription. Notez que les champs en jaune sont obligatoires, confirmez ensuite les conditions générales de vente et cliquez sur « Continuer ». Vous pouvez alors demander aussitôt une information cartographique.


Retour au sommaire

 


2. Enregistrement comme utilisateur permanent

Remplissez le formulaire d’inscription. Notez que les champs en jaune sont obligatoires et doivent être remplis. Définissez un mot de passe valide, confirmez ensuite les conditions générales de vente et cliquez sur « Continuer ». Dès que l’administrateur aura vérifié vos indications, votre compte sera validé. Vous serez informé par e-mail et vous pourrez alors commencer vos requêtes de données.


Retour au sommaire

 

3. Réinitialiser le mot de passe

Indiquez l’adresse e-mail utilisée pour votre compte, et cliquez sur « Réinitialiser mot de passe ». Vous recevrez alors un mail avec le nouveau mot de passe. Après votre connexion, vous avez la possibilité de modifier le nouveau mot de passe dans l’onglet Profil.

Retour au sommaire

 


4. Consultation de plan

Recherche d'adresse

Dans la recherche d’adresse, vous pouvez indiquer l’adresse souhaitée dans la liste à choix. Cliquez sur « Zoomer sur l’objet » afin de déplacer l’extrait de carte à l’emplacement souhaité.

Définition du cadre de plan

Sous Définition du cadre de plan, vous définissez le cadre plan souhaité pour l’information cartographique.

 

Dans la liste à choix « Format d’impression », sélectionnez le cadre de plan souhaité.

Définissez l’échelle du plan dans la liste à choix

Avec la souris, déplacez le cadre de plan directement sur la carte à la position souhaitée

Facultativement, vous pouvez définir une rotation du plan soit avec la réglette, soit directement en indiquant l’angle de rotation.

Pour naviguer dans la carte, vous pouvez utiliser les fonctions suivantes :

 

Agrandir l’extrait de carte

Réduire l’extrait de carte

Déplacer l’extrait de carte

Retour à l’extrait de carte précédent

Charger l’extrait de carte initial

Avancer à l’extrait de carte suivant

Activer et déplacer le cadre de plan

Déplacer le cadre de plan au centre de l’extrait de carte

Déplacer l’extrait de carte sur le cadre de plan

En outre, vous pouvez ajouter de nouveaux cadres de plans et supprimer des cadres de plans existants.

Ajouter un nouveau cadre de plan. Si cette fonction est grisée, soit aucun format d’impression n’est sélectionné, soit le nombre maximum de cadres de plans est atteint.

Supprimer le cadre de plan actif

 

Formulaire avec informations de commande

Remplissez le formulaire avec les informations de commande. Notez que les champs en jaune sont obligatoires et doivent être remplis. Confirmez les conditions générales de vente et cliquez sur « Commander ». Dès que les données seront disponibles, vous serez informé par e-mail. Vous recevrez alors un lien pour télécharger les données.

Retour au sommaire

 

 

5. Aperçu des requêtes (uniquement pour les utilisateurs permanents)

 

Cette page présente un aperçu de vos requêtes.

 

 

Recherche et filtrage des données

 

Sous « Recherche et filtrage des données », vous pouvez chercher des requêtes spécifiques. Le bouton « Rechercher » permet de filtrer la liste des résultats. Plusieurs critères de recherche peuvent être définis simultanément. Si aucun critère de recherche n’est défini, toutes vos requêtes s’affichent tant qu’elles n’ont pas été archivées par l’administrateur système.

 

 

Options

 

Sous « Options », la liste à choix permet de définir le nombre d’éléments par page. Les résultats de toutes les pages peuvent être exportés comme fichier CSV avec le bouton « Exporter ».

 

 

Tableau

 

Le tableau contient les requêtes avec leurs détails.

 

Au-dessus du tableau, le nombre de requêtes trouvées est indiqué.

 

Sous le tableau, les diverses pages de résultats peuvent être affichées  .

 

Un simple clic sur la colonne souhaitée permet de trier les données en fonction de cette colonne

 

La colonne « Nbre téléchargement » indique combien de fois les données ont été téléchargées.

 

La dernière colonne du tableau sert à télécharger les données :

 

    Les données peuvent être téléchargées. Il suffit de cliquer sur ce bouton.

 

    Les données ne peuvent plus être téléchargées, car le délai de téléchargement est écoulé.

6. Afficher et modifier les données du profil (uniquement pour les utilisateurs permanents)

 

Vos données

Sur cette page, vous pouvez consulter et modifier vos données d’utilisateur. Vous pouvez aussi redéfinir votre mot de passe.

Modification des données d’utilisateur :

Saisissez vos nouvelles données d’utilisateur et cliquez sur « Modifier les données d’utilisateur ». Notez que les champs en jaune sont obligatoires et doivent être remplis.

 

Modifier le mot de passe :

Saisissez votre ancien mot de passe et votre nouveau mot de passe. Le nouveau mot de passe doit être saisi dans 2 champs. Notez que votre nouveau mot de passe doit contenir au moins 5 caractères et se composer de lettres et de chiffres.
