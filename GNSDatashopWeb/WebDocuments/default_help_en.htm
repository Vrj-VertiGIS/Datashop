Welcome to GEONIS Datashop!

With GEONIS Datashop it is easy to use safe and simple data queries.

To request map requests, you have to register as occassional user or business user. If you do not have a business user account, please login as follows:

a.   If you need a map request very rarely, just use the occassional user account. As an occassional user, you can request five orders within two weeks.

b.   If you need map requests very often, we recommend to register as business user. Orders are not limited. Click on "Register as a new business user"

If you are already registered as business user, and if you have forgotten your password, click on "Lost password"?

 

1. Login as occassional user

Fill-in the registration form. Please note that the yellow fields are mandatory and have to be filled-in. Accept the terms and conditions and click on "Continue". You can obtain map requests directly.


Back to summary

 


2. Registration as a business user

Fill-in the registration form. Please note that the yellow fields are mandatory and have to be filled-in. Define a valid password, accept the terms and conditions and click on "Continue". As soon as the administrator has checked your details, your account can be used. You will be informed via e-mail. After this confirmation you can start querying data.


Back to summary

 

3. Reset password

Enter your e-mail address used for the account and click on "Reset password". You will receive a new password via e-mail. After a successful login, you can change the password in the tab "My Profile".

Back to summary

 


4. Map request

Address search

You can select the required address in the picklist. Click on "Zoom to object" to move the map extract to the required location.

Plot frame definition

In the plot frame section, the required plot frame can be defined for the map request.

 

In the picklist "Plot format", the required plot frame can be selected.

Define the map scale via the picklist.

Move the plot frame via mouse onto the required position.

The rotation angle is optional. You can define it via slider or by entering the angle manually.

For the map navigation, the following functions are available:

 

Zoom in

Zoom out

Pan

Previous map extent

Reset map extents to default settings

Next map extent

Move plot frame

Place plot frame in center of map

Zoom map on plot frame(s)

Additional plot frames can be added or deleted.

Add plot frame. If this function is grayed out, either no plot format has been selected, or the max. plot frame number has been reached.

Remove select plot frame

 

Form with ordering information

Fill-in the form with your ordering information. Please note that the yellow fields are mandatory and have to be filled-in. Confirm the terms and conditions and click on "Submit request" As soon as the data is available, you will be informed by e-mail. The e-mail contains a link with which you can download the data.

5. My Plots

 

On this page, you get an overview of your requests.

 

 

Search and filter data

 

Under "Search", you can find specific requests. With the button "Search", your results list will be filtered. Several search criteria can be defined. If no search criteria has been entered, all of your requests are shown (as long as they have not been archived by the system administrator).

 

 

Options

 

Under "Options", the number of elements per page can be defined. Via "Export2, the results of all pages can be exported as a CSV file.

 

 

Table

 

In the table, you see your requests with details.

 

On top left of the table, the number of found results is shown.

 

Beneive the table, the different result pages can be shown  .

 

By clicking on the column header, the data in the required column can be sorted.

 

"Download count" shows, how often the data has been downloaded.

 

With the last column, you can download the data:

 

    The data can be downloaded by clicking on this button.

 

    The data cannot be downloaded anymore, as the time of availability has expired.

 

leer    The request has not yet been processed by the system. The data is not yet available.

6. Show and change profile data (for business users only)

 

Your profile

On this page, you can review and change your user data. Your password can be redefined as well.

Change personal information:

Enter your new user data and click on "Change personal information". Please note that the yellow fields are mandatory and have to be filled-in.

 

Change password:

Enter your old and new password. The new password has to be entered in two fields. Please note to enter a password containing at least 5 characters (letters and numbers).
