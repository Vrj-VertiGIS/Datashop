Willkommen zu GEONIS Datashop!

Mit GEONIS Datashop können Sie sichere und einfache Datenabfragen ausführen.

Damit Sie Planauskünfte beziehen können, müssen Sie sich im System als Gelegenheitsnutzer oder als Dauernutzer registrieren. Wenn Sie noch kein Dauernutzerkonto besitzen, sollten Sie sich wie folgt anmelden:

a.    Brauchen Sie eine Werkleitungsauskunft nur gelegentlich, dann sollten Sie sich als Gelegenheitsnutzer anmelden. Als Gelegenheitsnutzer können Sie innerhalb von 2 Wochen 5 Bestellungen aufgeben.

b.    Brauchen Sie regelmässig Werkleitungsauskünfte, dann sollten Sie als Dauernutzer ein Konto eröffnen damit sie beliebig viele Bestellungen aufgeben können. Klicken Sie dafür auf „Neu registrieren“

Wenn Sie als Dauernutzer schon angemeldet sind und Sie ihr Passwort vergessen haben, klicken Sie auf „Passwort vergessen?“

 

1. Anmeldung als Gelegenheitsnutzer

Füllen Sie das Anmeldeformular aus. Beachten Sie, dass die gelbmarkierten Felder obligatorisch sind und unbedingt abgefüllt werden müssen, bestätigen Sie anschliessend die Geschäftsbedingungen und klicken Sie auf „Weiter“. Sie können dann direkt eine Planauskunft beziehen.


Zurück zur Zusammenfassung

 


2. Registrierung als Dauernutzer

Füllen Sie das Anmeldeformular aus. Beachten Sie, dass die gelbmarkierten Felder obligatorisch sind und abgefüllt werden müssen. Definieren Sie ein gültiges Passwort, bestätigen Sie anschliessend die Geschäftsbedingungen und klicken Sie auf „Weiter“. Sobald der Administrator Ihre Angaben überprüft hat wird Ihr Account freigegeben. Sie werden per E-Mail benachrichtigt und können anschliessend die Datenabfragen starten.


Zurück zur Zusammenfassung

 

3. Passwort zurücksetzen

Geben Sie die E-Mailadresse an, die für Ihr Konto benutzt wird und klicken Sie auf „Passwort zurücksetzen“. Sie bekommen anschliessend eine Mail mit dem neuen Passwort. Nach Ihrer Anmeldung, haben Sie im Reiter Profil die Möglichkeit Ihr neu erstelltes Passwort zu ändern.

4. Planauskunft

Adresssuche

In der Adresssuche können Sie die gewünschte Adresse in der Auswahlliste angeben. Klicken Sie auf „Zum Objekt zoomen“ damit der Kartenausschnitt an den gewünschten Ort verschoben wird.

Planrahmendefinition

Unter Planrahmendefinition wird der gewünschte Planrahmen für die Planauskunft definiert.

 

In der Auswahlliste „Druckformat“ wählen Sie den gewünschten Planrahmen aus

Legen Sie den Planmassstab in der Auswahlliste fest

Schieben Sie den Planrahmen mit der Maus direkt in der Karte auf die gewünschte Position

Optional, definieren Sie eine Planrotation entweder mit dem Schieberegler oder direkt mit Angabe des Rotationswinkels.

Zum Navigieren in der Karte können Sie folgende Funktionen benutzen:

 

Kartenausschnitt vergrössern

Kartenausschnitt verkleinern

Kartenausschnitt verschieben

Zurück zum vorherigen Kartenausschnitt

Initial Kartenausschnitt laden

Vor zum nächsten Kartenausschnitt

Planrahmen aktivieren und verschieben

Planrahmen ins Zentrum des Kartenausschnitts verschieben

Kartenausschnitt auf Planrahmen verschieben

Ausserdem können Sie neue Planrahmen hinzufügen oder bestehende Planrahmen löschen.

Neuer Planrahmen hinzufügen. Ist diese Funktion ausgegraut, ist entweder kein Druckformat ausgewählt oder die maximale Planrahmenanzahl erreicht.

Aktiver Planrahmen löschen

 

Formular mit Bestellungsinformationen

Füllen Sie das Formular mit den Bestellungsinformationen aus. Beachten Sie dabei, dass die gelbmarkierten Felder obligatorisch sind und unbedingt abgefüllt werden müssen. Bestätigen Sie die Geschäftsbedingungen und klicken Sie auf „Bestellen“. Sobald die Daten zur Verfügung stehen, werden Sie per E-Mail benachrichtigt. Sie erhalten dann einen Link mit dem Sie die Daten herunterladen können.

5. Anfragenübersicht (nur für Dauernutzer)

 

Auf dieser Seite erhalten Sie eine Übersicht Ihrer Anfragen.

 

 

Suchen und Filtern der Daten

 

Unter „Suchen und Filtern der Daten“ können Sie nach spezifischen Anfragen suchen. Mit dem Knopf „Suchen“ wird die Ergebnisliste gefiltert. Es können gleichzeitig mehrere Suchkriterien definiert werden. Wird kein Suchkriterium angegeben, so werden alle Ihre Anfrage angezeigt, solange sie vom Systemadministrator nicht archiviert wurden.

 

 

Optionen

 

Unter „Optionen“ können mit der Auswahlliste die Anzahl Elemente pro Seite definiert werden. Die Ergebnisse von allen Seiten können mit dem Knopf „Exportieren“ als CSV-Datei exportiert werden.

 

 

Tabelle

 

In der Tabelle werden die Anfragen mit den Anfragendetails angezeigt.

 

Oberhalb der Tabelle ist die Anzahl gefundener Anfragen ersichtlich.

 

Unterhalb der Tabelle können die verschiedenen Ergebnisseiten angezeigt werden  .

 

Mit einem Klick auf die gewünschte Spalte können die Daten nach dieser Spalte sortiert werden

 

In der Spalte „Anzahl Download" steht wie oft die Daten heruntergeladen wurden.

 

Die letzte Spalte der Tabelle dient zum Herunterladen der Daten:

 

    Die Daten können herunterladen werden. Dies geschieht mit Klick auf diesem Button.

 

    Die Daten können nicht mehr herunterladen werden, da die Herunterladefrist abgelaufen ist.

 

leer    Die Anfrage wurde noch nicht vollständig vom System bearbeitet und die Daten sind noch nicht verfügbar.

6. Profildaten anschauen und ändern (nur für Dauernutzer)

 

Ihre Daten

Auf dieser Seite können Sie Ihre Benutzerdaten nachsehen und ändern. Ihr Passwort kann ebenfalls neu definiert werden.

Benutzerdaten ändern:

Geben Sie Ihre neue Benutzerdaten ein und klicken Sie auf „Benutzerdaten ändern“. Beachten Sie, dass die gelbmarkierte Felder obligatorisch sind und unbedingt abgefüllt werden müssen.

 

Passwort ändern:

Geben Sie Ihr altes Passwort und Ihr neues Passwort an. Das neue Passwort muss in 2 Felder angegeben werden. Beachten Sie, dass Ihr neues Passwort mindestens 5 Zeichen lang ist und dass es aus Buchstaben sowie Zahlen besteht.
